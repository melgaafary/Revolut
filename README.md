# Getting started

    npm install
    npm start
should run http://localhost:8080

# Tests

    npm run tests

## Stack used
- React
- React hooks  - for the state
- Grommet - for UI components
- Jest/enzyme
- Eslint and airbnb config
- [Open exchange for the rates](https://openexchangerates.org/)
